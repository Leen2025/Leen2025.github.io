الرئيسة الاستفسارات

البنية التحتية (IT Infrastructure)

وش هو؟
هو الأساس اللي يشتغل عليه كل شي تقني! زي الشبكات، السيرفرات، الأجهزة، وأمنها.

وش يسوّون فيه؟
يركبون الشبكات، يديرون السيرفرات، يصلّحون الأعطال، ويتأكدون أن كل شي يشتغل تمام.

خطة البداية:

  • تعلّم مفاهيم الشبكات (مثل OSI Model)
  • خذ دورات في CCNA و+Network
  • جرب إعداد بيئة افتراضية (VirtualBox / VMware)
  • تعلّم عن السيرفرات ونظام Linux
  • شارك في مشاريع شبكات أو صيانة

مصادر قوية ومجانية لتعلم البنية التحتية التقنية (IT Infrastructure)

دورات مجانية وقوية:

  • Google IT Support – Coursera
    رابط الدورة
    مقدمة شاملة ومناسبة للمبتدئين، تشمل الشبكات، أنظمة التشغيل، إدارة النظام، وخدمات الدعم الفني.
  • Introduction to Systems Administration – Udacity
    رابط الدورة
    تعلم إدارة الأنظمة، السيرفرات، إعدادات الشبكة، والمزيد.
  • Networking Fundamentals – Cisco Networking Academy
    رابط الدورة أساسيات الشبكات من شركة Cisco، مثالية لفهم الإنترنت وكيفية اتصال الأجهزة ببعضها.
  • Linux Essentials – edX by Linux Foundation
    رابط الدورة
    مدخل مهم لأنظمة لينكس، النظام الأكثر استخدامًا في البنية التحتية.

كتب ممتازة لتعلم البنية التحتية :

  • The Practice of System and Network Administration – Thomas Limoncelli
    مرجع كلاسيكي يغطي مفاهيم إدارة الأنظمة والشبكات باحترافية.
  • Networking All-in-One For Dummies – Doug Lowe
    شرح بسيط وواضح لكل ما يخص الشبكات.
  • Linux Command Line and Shell Scripting Bible – Richard Blum
    تعلم أوامر لينكس والبرمجة النصية لإدارة الخوادم والأنظمة.

مواقع وأدوات تعليمية قوية:

  • Cisco Networking Academy
    netacad.com
    أقوى منصات تعليم الشبكات والشهادات الاحترافية مثل CCNA.
  • Red Hat Developer
    developers.redhat.com
    دروس وأدوات لتعلم إدارة الخوادم وLinux والتقنيات السحابية.
  • Microsoft Learn
    learn.microsoft.com
    دورات مجانية من مايكروسوفت عن Azure، الخوادم، الشبكات، والحوسبة السحابية.
  • Pluralsight – Infrastructure Paths
    openai.com/learn
    دورات متخصصة للبنية التحتية، بعضها مجاني مع فترة تجريبية. .

أدوات تساعدك تطبق وتتعلم أسرع:

  • VirtualBox / VMware
    محاكيات لإنشاء بيئات شبكية واختبار الأنظمة افتراضيًا.
  • GNS3 / Cisco Packet Tracer
    محاكيات شبكات لبناء شبكات افتراضية والتجربة عليها.
  • Ansible / Terraform
    أدوات لإدارة البنية التحتية كرمز (Infrastructure as Code).

خطة مقترحة لك:

  1. ابدأ بدورة Google IT Support لفهم الأساسيات.
  2. تعلم Linux من دورة Linux Foundation.
  3. خذ دورة Cisco في أساسيات الشبكات.
  4. طبق عمليًا باستخدام VirtualBox أو Packet Tracer.
  5. خذ مسار Microsoft Azure أو AWS للبنية السحابية.
  6. تدرب على أدوات Infrastructure as Code مثل Ansible.
  7. تابع مجتمعات Tech Infrastructure على LinkedIn وX.
